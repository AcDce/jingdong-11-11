# jingdong-11-11
京东双11，全民营业，瓜分10亿活动

# 测试环境
蓝叠模拟器3.1.21 + autojs4.1.1 + 京东v9.2.0

# 脚本功能列表
 - 自动执行-8秒浏览任务
 - 自动执行-浏览5个商品任务
 - 自动执行-加购5个商品
 - 自动执行-简单任务
 - 自动执行-精选联合会员任务

# 使用说明
 1. 下载 [autojs4.1.1](https://share.weiyun.com/owu3tBNr)，并开启无障碍服务，导入上面脚本
 2. 打开京东双11活动主页面，运行该脚本
